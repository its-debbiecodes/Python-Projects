## Number Guessing Game #️⃣
[Number guessing game.py](https://github.com/its-debbiecodes/Python-Projects/blob/main/Number%20guessing%20game.py)

### Description
This is a program that picks a number at random beween 1 and 100, asks the user for an input of a guessed number, gives them a hint if thier guess is less or higher than the secret number, it then prints their results if right or wrong and how many attempts it took.

### Features
- User input
- random import from 1 - 100
- if satements for hints
- while loop for running the user input again
- play again statemens
- Boolean for automating a new game

### How to Run
- Make sure Python is installed
- Download the Number guessing game.py
- Run the program in the terminal
